Documentation

lightning-accordionmodule

A collection of vertically stacked sections with multiple content areas.

Support:GA
Access:global
Abstract:false
Extensible:false

Attributes

NameTypeAccessRequiredDefaultDescription
active-section-nameglobalExpands the specified accordion sections. Pass in a string for a single section or a list of section names. Section names are case-sensitive. To support multiple expanded sections, include allow-multiple-sections-open in your markup. By default, only the first section in the accordion is expanded.
allow-multiple-sections-openglobalIf present, the accordion allows multiple open sections. Otherwise, opening a section closes another that's currently open.
titleglobalDisplays tooltip text when the mouse moves over the element.

NameDescription
